# 프로젝트 설명
## 사용 스택
- React
- TypeScript
- Firebase(로그인, 보안, 통신, 이미지)
- Emotion
- Recoil
- Vercel
## 주요 기능
- 로그인 및 회원가입
- InfiniteScroll 카드 리스트
- 카드 신청 및 해지
- 신청한 카드 목록
- 프로필 이미지 등록 및 수정
## 앱 구조
- Single Page Application (SPA)
## 상태관리
- Recoil
- 권한관리
